About Me


Hi there! My name is Cari. Thanks for checking out my game, Cairdeas Quest. I designed this game as part of my Master's Degree Programme at the National University of Ireland, Galway. Cairdeas Quest aims to teach children to identify bullying and properly react in bullying situations.Â 

What Data does Cairdeas Quest Collect?

Cairdeas Quest collects the following data: players's age, game start time, game end time, hearts collected, and how many times it takes the player to answer each question correctly. No identifying data is collected.
